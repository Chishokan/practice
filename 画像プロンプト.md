# キャラクター画像 生成プロンプト集

## 使い方

1. 下のプロンプトを Claude / GPT / 画像生成AIに渡す
2. 出てきた画像を `練習フォルダ/images/` に、指定のファイル名で保存する
3. ゲームを再読み込みすると、部員カードと詳細ページに反映される

画像がなくても動きます。その場合は頭文字が表示されます。

---

## 共通のスタイル指定

**全キャラ共通。必ずプロンプトの先頭に付けること。**
10人が並んだときに画風がバラバラだと台無しになるので、ここが一番大事です。

### 日本語版

```
高校バスケットボール漫画のキャラクターデザイン。
上半身のバストアップ、正面やや斜め、視線はこちら。
画風は90年代〜2000年代の少年漫画寄りのアニメ塗り。線は太めでくっきり。
背景は透過（または濃紺 #12151c の単色）。
服装はチームのバスケユニフォーム：濃紺(#1c212c)地にオレンジ(#ff7a2f)のライン、
胸に「陽和(ひより)高校」のロゴ、背番号は各キャラの指定に従う。
光源は左上からの体育館の窓明かり。影はやや硬め。
10人全員が同じ画風・同じ塗り・同じ画角で並ぶことを最優先にすること。
正方形 1:1、512×512px、PNG。
```

### 英語版（海外モデル向け）

```
Character design for a Japanese high school basketball manga.
Bust-up portrait, front-facing slightly angled, looking at viewer.
Style: late-90s/2000s shonen manga anime cel-shading, bold clean linework.
Background: transparent (or flat dark navy #12151c).
Outfit: team basketball uniform, dark navy (#1c212c) with orange (#ff7a2f) trim,
"陽和" school logo on chest, jersey number as specified per character.
Lighting: gymnasium window light from upper left, fairly hard shadows.
CRITICAL: all 10 characters must share identical art style, shading, and framing.
Square 1:1, 512x512px, PNG.
```

### デザインの決まりごと

苗字に色が入っている選手は、**その色を髪のアクセントに使う**。
赤星＝赤、青木＝青、黒田＝黒、白石＝白、緑川＝緑、紫藤＝紫、桃井＝桃、灰谷＝灰。
ただし彩度は抑えること。並べたときにチームとして見えるようにするため。

## エンディングの1枚絵（4種）

1年の終わり（ウィンターカップ終了時）に、結果に応じて1枚だけ出ます。
バストアップではなく**横長の情景画**なので、共通スタイルではなく下の指定を使ってください。

### エンディング共通の指定

```
高校バスケットボール漫画のクライマックスの1枚絵。
横長 16:9、1280×720px、PNG。
画風は90年代〜2000年代の少年漫画寄りのアニメ塗り。線は太めでくっきり。
キャラクター表と同じ画風・同じ人物デザインを保つこと。
画面の下3分の1は文字を載せるので、暗めにして、要素を置かないこと。
チームカラーは濃紺(#1c212c)＋オレンジ(#ff7a2f)。
```

## メインストーリーの1枚絵（9話）

毎月1話ずつ進むメインストーリーの、各話の扉に出る**横長の情景画**です。
ノベルのページ送りの上に、バナーとして表示されます（文字は絵の上には載りません）。
画像が無い場合は、話のタイトルを載せた枠が代わりに出るので、無くても動きます。

同じ話でも分岐（勝ち／負けなど）が複数ありますが、1枚を共有します。
そのため、どの分岐でも矛盾しない「その月に共通の情景」を指定しています。

### ストーリー共通の指定

```
高校バスケットボール漫画の、章扉の1枚絵。横長 16:9、1280×720px、PNG。
画風は90年代〜2000年代の少年漫画寄りのアニメ塗り。線は太めでくっきり。
キャラクター表と同じ画風・同じ人物デザインを保つこと。
チームカラーは濃紺(#1c212c)＋オレンジ(#ff7a2f)。舞台は古びた高校の体育館の半面。
文字は載せないので、画面いっぱいに情景を描いてよい。
```
