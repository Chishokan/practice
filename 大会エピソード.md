# 大会エピソード（校正用）

大会の節目に出る短い話です。メインストーリーと同じ**ページ送りのノベル**で表示されます。

- 「— ページN —」がページの区切りです。ここで読み手が「次へ」を押します。
- **決勝後**の話は、決勝まで辿り着いた年だけ出ます（早期敗退の年には出ません）。
- 1枚絵は付けていません。付けたい場合は各話に画像を割り当てられます。
- この文面を直接書き換えてください。そのまま反映します。
